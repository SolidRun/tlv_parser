# TLV CMD TOOL

A command line tool to parse TLV data from EEPROM.

### Compilation
For native compilation, just run:
```
make
```
For cross compilation, you can specify the compiler to use:
```
COMPILER=<your-compiler> make
```

### Installing
Just run ```make install``` with the destination path, for example:
```
INSTALL_DIR=/bin/ make install
```

### Usage
Call the program without arguments to see usage menu.
